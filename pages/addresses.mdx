# Addresses

This is a list of well known EOA and Smart Contract addresses used and published by Grassroots Economics. A detailed documentation on our smart contract interfaces is available [here](https://git.grassecon.net/cicnet/cic-contracts/).

We provide a [reference implementation in Go](https://pkg.go.dev/github.com/grassrootseconomics/ethutils) on how to easily load addresses described below.

<Callout type="info" emoji="ℹ️">
  All Smart Contracts published by Grassroots Economics are **always verified** on Celoscan.
</Callout>
